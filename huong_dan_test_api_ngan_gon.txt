HUONG DAN TEST API SMART STAY (BAN NGAN GON)

1) CHAY SERVER
- Mo terminal tai thu muc chua file.
- Chay lenh:
  uvicorn a00main:app --reload
- Sau do mo trinh duyet:
  http://127.0.0.1:8000/docs

2) CACH TEST TREN /docs
- Bam vao API can test.
- Bam nut Try it out.
- Neu la GET: bam Execute de chay.
- Neu la POST: dan noi dung mau vao o Request body, roi bam Execute.
- Xem phan Response body de coi ket qua.

3) Y NGHIA CHUNG
- GET = dung de xem / hien thi du lieu, hoac kiem tra he thong va database co dang hoat dong binh thuong khong.
- POST = dung de gui noi dung nguoi dung nhap vao de he thong loc, de cu, hoac chatbot tra loi.

========================
LOI THUONG GAP
========================

1. Neu vao /docs ma khong thay du cac API:
- Kiem tra lai file a00main.py.
- Neu bi tao app = FastAPI() 2 lan thi co the mat router.

2. Neu GET /test-db hoac GET /health bao loi:
- Kiem tra SQL Server da chay chua.
- Kiem tra database SmartStayDB co ton tai khong.
- Kiem tra chuoi ket noi trong config.

3. Neu POST bi loi 422:
- Thuong la dan sai body JSON.
- Can dan dung mau nhu tren.

4. Neu API tra ve rong:
- Co the database chua co du lieu phu hop.

========================
TOM TAT CUC NGAN
========================
- GET: de xem / hien thi du lieu, va kiem tra backend + database co hoat dong binh thuong khong.
- POST: de gui noi dung nguoi dung nhap vao.
- Cach test nhanh nhat: mo http://127.0.0.1:8000/docs -> bam Try it out -> nhap body neu can -> bam Execute.
